ConcentMeter
============
集中しなければいけない時についついサボり心がでてしまうアナタにオススメです。
HVC-Cを正面においてアプリを起動するだけで、どのくらいの時間集中しているかを
リアルタイムにメーターで表示します。よそ見をするとどんどん減っていくので
( ﾟдﾟ)ﾊｯ!と気づく事ができます。

OmronのHVC-CデバイスとSDKが必要です。
